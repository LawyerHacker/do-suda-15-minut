# До суда 15 минут

> Юрист опаздывает на заседание. До суда — пятнадцать минут, три дорожки и ни одной права на ошибку.

«До суда 15 минут» — трёхмерный аркадный раннер с юридической сатирой. Игрок проводит героя через офис, городскую улицу и коридоры арбитражного суда: уворачивается от препятствий, прыгает, делает подкаты и собирает бусты уверенности, чтобы успеть к началу заседания.

## Что внутри

- Три уровня с разным настроением: офис, город и суд.
- Простое, быстрое управление: три дорожки, прыжок и подкат.
- Система уверенности: удачные находки усиливают героя, ошибки снижают запас сил.
- Юридические шутки, редкие суперпризы и «токсичные» препятствия.
- Стильная low-poly графика: большая часть мира, эффектов и интерфейса собирается процедурно из C#-кода.
- Версии для Windows и WebGL; в браузерной версии работают клавиатура и свайпы на телефоне.

Игра не связана официально с судами, Росреестром или другими государственными органами. Все игровые ситуации и персонажи вымышлены.

## Быстрый запуск в Windows

Чтобы играть без Unity, скачайте репозиторий через **Code → Download ZIP**, распакуйте архив целиком и запустите `Build/CourtRunner15.exe`.

Важно: не переносите `.exe` отдельно — рядом с ним должна остаться папка `CourtRunner15_Data` и остальные файлы из `Build`.

## Запуск проекта

Проект создан в **Unity 2022.3.10f1 LTS**.

1. Установите эту версию через Unity Hub.
2. В Unity Hub нажмите **Add / Открыть** и выберите папку проекта.
3. Дождитесь импорта и компиляции скриптов.
4. Откройте сцену `Assets/CourtRunner15/Scenes/Main.unity` и нажмите **Play**.

Если сцена не открылась, выберите меню **«До суда 15 минут → Создать или обновить сцену»**. Игра также умеет запускаться из пустой сцены: bootstrap создаёт нужные объекты сам.

## Управление

| Клавиша | Действие |
|---|---|
| A / ← | Левая дорожка |
| D / → | Правая дорожка |
| Space / W / ↑ | Прыжок |
| S / ↓ / Shift | Подкат |
| Esc | Пауза / назад |
| R | Перезапуск уровня |
| 1 / 2 / 3 | Быстрый запуск уровня на экране выбора |

На телефоне используйте свайпы: влево и вправо для смены дорожки, вверх для прыжка, вниз для подката. Игра рассчитана на горизонтальную ориентацию экрана.

## Правила

- Стартовая уверенность — 60%.
- Бусты дают уверенность, столкновения её отнимают.
- При уверенности от 75% герой ускоряется; от 90% получает защиту от одной ошибки.
- Ноль уверенности означает поражение, пройденная дистанция — победу.

## Проверка проекта

В Unity используйте **«До суда 15 минут → Проверить проект»**, либо запустите проверку из командной строки:

```powershell
Unity.exe -batchmode -projectPath "<путь к проекту>" -executeMethod CourtRunner15.EditorTools.CourtRunnerValidator.RunBatch -quit -logFile validate.log
```

## Состав репозитория

В репозитории лежат исходники, игровые ассеты и настройки Unity. Кэши Unity, логи, локальные сборки и служебные заметки исключены через `.gitignore`: они не нужны для сборки проекта и не содержат исходного кода игры.
